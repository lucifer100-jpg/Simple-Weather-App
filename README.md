# Weather-recipe-app

##Overview
A simple Front-end web application that helps select food based on the weather. It uses static data without any real APIs.
It is made using HTML, CSS, Javascript with static JSON.

##How to run the app
1. Download or copy simpleweatherapp.html file.
2. Save it on your computer.
3. Open it with chrome.
4. The recipe will load according to the weather.

##Assumptions Made
There is no real APIs so the weathers should be inserted manually. Each weather has a suitable recipe. There are only 4 weather types inserted being(Cold, Hot, Rainy, Mild). There are already predefined recipe given.

##Notes
It shows how ststic JSON data can be used in a front-end application without external APIs or backend services.
